# Button Component

A versatile button component built on top of PrimeNG's button, providing various styles, sizes, and accessibility features.

## Key Features

- Multiple visual styles through the `type` property
- Three size options: small, normal, and large
- Icon support with customizable positioning
- Built-in accessibility features
- Tooltip support

## Accessibility

The button component follows accessibility best practices:

- Proper ARIA labels for icon-only buttons
- Keyboard navigation support
- Focus management
- Optional tooltips for additional context
